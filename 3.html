CRESTWORTH MANOR

 

CREATED BY NUPPY, NEDEAJJ

MAP INFO

This is a house me and my friend created. This is an old family-owned house with an ominous nature behind it.

It is meant to be played as a exploration map, so feel free to do what you want with it, however, we created it so that you can explore it freely. It has a spooky nature to it and quite honestly, with the hours we spent on it, it really does get pretty ominous after sometime of being on the world.

Just a few more extra notes:
Read the rules in the beginning of the map. This map is multiplayer compatible, just make sure you stay close with your friend, and that you guys walk through the map together. If you spawn in a random location for some strange reason, the coordinates to the house are: 90057 30 90057. We have added a reset button on the map to reset the command blocks, but just know that it does not reset everything. We have included a few easter eggs in the map as well, there are two secret rooms, one secret passageway, and one secret book. These secrets are not important they are just side things.

The world file is going to appear as "world".
